Como dar Estilo a una página web

Para dar estilo a una página web se utiliza el lenguaje de marcado CSS, con html se puede dar estilo a los sitios web, sin embargo es más efectivo utilizar CSS.

Paso 1

Hay dos formas de utilizar CSS, la primera es usar el atributo “style” en HTML5, la segunda es hacer otro documento por aparte y enlazarla con el documento de HTML, es muy aconsejable usar la segunda forma ya que de esta manera se puede volver a usar a en otros documentos. Entonces se abre el editor de texto.

Paso 2

CSS no es un lenguaje muy complicado, existe la regla de estilo que consiste en tres partes: selector, propiedad y valor.


Paso 3

Entonces con el selector, seleccionamos el elemento que será editado, por ejemplo, h1 que es un titulo con la propiedad Font-style y asignando el valor italic, el titulo aparecerá de forma cursiva.


Nota

Existen varias propiedades con diferentes valores, para que tú página tenga más estilo, es recomendable investigar las propiedades con sus respectivos valores.

Nota

Al seleccionar una etiqueta las demás etiquetas con el mismo nombre serán afectados, por ejemplo, si secciono la etiqueta “p”(de párrafo) todas las etiquetas “p” serán afectadas.

Paso 4

Existen otros tipos de selectores como lo son las clases y los id, estos dos selectores solo afectan a un grupo en específico, en el siguiente ejemplo se utiliza las clases en html se escribe el atributo class y en css se asigna con “.” y el nombre de la clase.


Paso 5

Para que funcione el diseño hecho se debe guardar el documento con la extensión “.css”, para que el sistema reconozca que tipo de formato es.

Paso 6

Ahora es necesario enlazarlo con el documento de html, entonces dentro de la etiqueta head, se crea una nueva etiqueta llamada link.

Paso 7

Ahora se crea un nuevo atributo de link llamada “rel” y se asigna el valor de “stylesheet”.

Paso 8

Por último, se crea otro atributo de link llamado “href” y el valor que se le asigna es la dirección de nuestro documento por ejemplo “estilo.css”
